# Peggy Design System Export

Complete style guide and design system deliverables for the Peggy-inspired Android app.

## Contents

| Folder | Description |
|--------|-------------|
| `design-tokens/` | CSS, Android XML, and Jetpack Compose color/spacing/shape tokens |
| `component-spec/` | Markdown component specification with usage & interaction rules |
| `accessibility/` | WCAG contrast audit and focus/hover accessibility report |
| `ci-cd/` | GitHub Actions + GitLab CI accessibility automation |
| `android/` | Full Jetpack Compose theme (Color.kt, Type.kt, Shape.kt, Theme.kt) |
| `web-src/` | Current TanStack Start source for the style guide page |
| `assets/` | Doodle illustrations (PNG) |

## Brand Summary

- **Personality**: Warm, playful, trustworthy — "made by a parent, for parents"
- **Primary font**: Plus Jakarta Sans (body), Fraunces italic (display emphasis)
- **Primary color**: Peggy Blue `#7DA1FF` (cornflower)
- **Ink**: Deep navy `#001A33` for all text and doodle strokes
- **Accent**: Sunshine Yellow `#FFD84D`, Coral `#F0445B`, Mint `#C9EFC3`, Amber `#BD814B`
